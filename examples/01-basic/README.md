# 基础使用示例

本示例展示了线程安全抽奖系统的基础功能使用方法。

## 功能演示

1. **Redis连接设置** - 如何连接和配置Redis
2. **抽奖引擎创建** - 创建基础的抽奖引擎实例
3. **范围抽奖** - 在指定数值范围内进行抽奖
4. **奖品池抽奖** - 基于概率的奖品池抽奖
5. **连续抽奖** - 一次性进行多次抽奖

## 运行示例

```bash
cd examples/01-basic
go run main.go
```

## 前置条件

- Redis服务器运行在 `localhost:6379`
- Go 1.19+ 环境
